# 📸 InstaGrid

인스타그램 피드를 위한 그리드 이미지 분할 도구

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen)](https://insta-grid.pages.dev)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Built with Amazon Q](https://img.shields.io/badge/Built%20with-Amazon%20Q-FF9900?logo=amazon-aws)](https://aws.amazon.com/q/)

## ✨ 주요 기능

- 🎨 **직관적인 편집**: 드래그로 이미지 이동, 마우스 휠로 확대/축소
- 📱 **모바일 지원**: 터치 제스처로 편집 가능
- 🎯 **정확한 비율**: 인스타그램 피드에 완벽하게 맞는 3:4 비율
- 📐 **다양한 그리드**: 1줄(3개), 2줄(6개), 3줄(9개) 선택 가능
- 🌈 **배경 자동 채우기**: 블러 처리된 배경으로 자연스러운 확장
- 💾 **간편한 다운로드**: 개별 또는 전체 다운로드 지원
- 🚀 **빠른 처리**: 클라이언트 사이드 처리로 즉시 생성

## 🎯 사용 방법

1. **이미지 업로드**
   - "이미지 선택하기" 버튼 클릭
   - 원하는 이미지 선택

2. **그리드 설정**
   - 캔버스 옆 [+] 버튼으로 줄 추가 (최대 3줄)
   - [-] 버튼으로 줄 제거 (최소 1줄)
   - 1줄: 3개 이미지 / 2줄: 6개 이미지 / 3줄: 9개 이미지

3. **이미지 편집**
   - 드래그로 위치 조정
   - 마우스 휠로 확대/축소
   - 모바일: 터치로 조작

4. **이미지 생성**
   - "이미지 생성하기" 버튼 클릭
   - 선택한 줄 수만큼 이미지 자동 생성

5. **다운로드 & 업로드**
   - 개별 또는 전체 다운로드
   - 인스타그램에 **역순**으로 업로드 (오른쪽 아래부터 왼쪽 위로)

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Canvas API**: 이미지 처리 및 편집
- **Design**: Glassmorphism, Dark Mode
- **Development**: Amazon Q Developer (Vibe Coding)
