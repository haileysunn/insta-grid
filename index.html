InstaGrid

인스타그램 연결된 피드를 위한 그리드 이미지 분할 도구

🔒 모든 처리는 브라우저에서 진행되며, 이미지는 서버에 저장되지 않습니다.

📱 앱으로 설치하기

홈 화면에 추가하여 앱처럼 사용하세요!

🤖 Android (Chrome)
  1. 주소창 옆 "설치" 버튼 클릭
  2. "설치" 확인
  3. 홈 화면에 아이콘 추가됨 ✨
🍎 iOS (Safari)
  1. 하단 공유 버튼 (⬆️) 탭
  2. "홈 화면에 추가" 선택
  3. "추가" 탭
  4. 홈 화면에 아이콘 추가됨 ✨
💻 Desktop (Chrome/Edge)
  1. 주소창 오른쪽 설치 아이콘 클릭
  2. "설치" 확인
  3. 독립 앱으로 실행됨 ✨

💡 오프라인에서도 사용 가능 | 빠른 로딩 | 앱처럼 실행

📱 사용법
• 한 손가락: 드래그로 이동
• 두 손가락: 핀치로 확대/축소

🖱️ 사용법
• 드래그: 이미지 이동
• 마우스 휠: 확대/축소

🎉 생성 완료!
